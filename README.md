# CleanTip – NRL Tipping App

CleanTip is a simple, ad-free NRL tipping app designed for users who want to make weekly NRL tips without the clutter, complexity, or sports-betting advertising found in other tipping platforms.

## Problem

Many existing sports tipping apps are bulky, difficult to navigate, and include gambling-related advertisements. This can be distracting or triggering for users who simply want a clean tipping experience with friends.

## Target Audience

Our target user is a casual NRL fan who enjoys tipping competitions but wants a simple, safe, and distraction-free app experience.

Example persona:

Ali is a university student who enjoys NRL tipping with friends. He wants to quickly view weekly fixtures, select his tips, and track results without being exposed to gambling content or unnecessary app clutter.

## App Goal

The goal of CleanTip is to provide a minimum viable product for a clean NRL tipping experience.

Users can:

- View NRL matches
- Select a winning team for each match
- Save and view their tips
- View completed results
- Track basic tipping performance
- View a simple leaderboard

## Key Features

### Match List

Displays upcoming NRL matches with team names, dates, and tip status.

### Make a Tip

Users can select which team they think will win each match.

### Results

Completed matches show final scores and whether the user’s tip was correct.

### Leaderboard

Displays player scores in a simple ranking format.

### Ad-Free Design

The app intentionally avoids betting advertisements and unnecessary distractions.
